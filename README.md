### 🔥 What I've Been Building Lately

Here’s what I’ve been coding up recently. No toy projects — just high-impact, scalable builds:

*   ⚡ **Energy 7 (E7) Pipeline**
    Dropped a β-VAE anomaly detector on 200+ railway machines. Squeezed signal compression to 99%+ with crazy fast ETL speeds. (*Python, FastAPI, Signal Processing*)

*   🧠 **[InnerTone](https://github.com/Shreyxpatil/InnerTone)** 
    A web-based mental wellness app featuring a fully interactive 3D browser avatar, voice AI, and a crisis-detecting RAG pipeline. (*React Three Fiber, Gemini API, WebRTC*)

*   💼 **[SideKick](https://github.com/Shreyxpatil/SideKick)** 
    My personal job-hunt cheat code. A Chrome extension that auto-fills applications across 10+ portals using Python scrapers. (*Chrome Ext API, Web Scraping*)

*   🎓 **GateTutor AI Platform** 
    A massive production ed-tech system: RAG MCQ generation, hybrid FAISS chatbots, and 95% accurate code plagiarism detection. (*Flask, OpenAI API, Docker*)

*   🗣️ **[Fish-Speech Inference Engine](https://github.com/Shreyxpatil/fish-speech_inference_)** 
    An end-to-end Python pipeline for cloning voices in 13 languages, completely bypassing the default CLI workflow. (*PyTorch, VQGAN, Audio DSP*)

*   👁️ **[Fast Hindi OCR Extractor](https://github.com/Shreyxpatil/fast-hindi-ocr-extractor)** 
    A hyper-optimized text extractor parsing 550+ complex scanned Hindi documents in under 15 minutes (85%+ accuracy). (*EasyOCR, Tesseract, OpenCV*)

*   📊 **[Tesseract 5 CAPTCHA Fine-Tuning](https://github.com/Shreyxpatil/tesseract5-captcha-finetuning)** 
    Pushing Tesseract 5 CAPTCHA recognition to its breaking point with custom data augmentation and hard-negative mining. (*Tesseract 5, LSTM*)

*   🤖 **Adaptive Text-to-SQL (Qwen 2.5)** 
    Hooked up Qwen 2.5 Instruct with LoRA to auto-generate brutal SQL queries across a 1M+ dataset while saving major GPU hours. (*HuggingFace, LoRA/QLoRA*)
